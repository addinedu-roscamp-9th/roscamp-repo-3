## Environment

Create venv at current jetcobot/ directory

```sh
python3 -m venv .venv
```

Activate the virtual environment

```sh
. .venv/bin/activate
```

Update pip

```sh
pip install -U pip
```

Install dependencies

```sh
pip install -r requirements.txt
```
