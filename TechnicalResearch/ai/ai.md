## Virtual Environments

Create venv named `.venv`

```sh
python3 -m venv .venv
```

Start the venv

```sh
. .venv/bin/activate
```

Update `pip`

```sh
pip install -U pip
```

Get the environment ready with `requirements.txt`:

```sh
pip install -r requirements.txt
```
